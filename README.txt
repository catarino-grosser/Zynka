REVISTA DIGITAL V1

Como usar:
1. Envie os arquivos index.html, style.css e app.js para o mesmo projeto no SPCK Editor.
2. Abra o index.html para testar.
3. Faça deploy no Netlify.

Sobre as notícias:
- Esta versão usa a API pública do GDELT.
- O app exibe título, fonte, data, imagem quando disponível e link para a matéria original.
- Não copia matérias completas, para reduzir risco de violação de direitos autorais.

Você pode editar no app.js as categorias e palavras de busca.
